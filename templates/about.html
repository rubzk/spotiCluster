How it works?

TL;DR

A web app that gather your data and create K-means clusters with it

Deep dive

Getting the data

This Flask web app uses the official Spotify Api to gather your playlists and then get the audio features of each song.

For parallel processing it uses Celery Workers

Clustering

After collecting all your data, it uses the K-means algorithm to create different clusters with your data.

How the algorithm works?

It uses the data points of the different audio features and assigned random centroids based on how many clusters do you want. Then it meausres the distance between the data points and the centroids and assigned the data point to the nearest cluster.

As a result, the algorithm calculates the mean of each cluster but to get the better cluster since the algorithm can see, it uses the variation of each iteration to decide the best cluster.

I recommend watching Staquest video if you want to have a deeper explanation

Plotting

The plots are made with my favorite plotting library Plotly which let me create the figures in Python and then encode them to be plotted in Js.
